aern2-mp-0.1.0.0: Multi-precision floats via haskell-mpfr or hmpfr

A wrapper around MPFR (and, in future, an alternative to MPFR), providing the types:

  • Dyadic: variable-precision floats with exact ring operations
  • MPFloat: fixed-precision floats with approximate field & elementary operations
  • MPBall: float ± error bound with field & elementary (interval-like) operations

By default, we use hmpfr when compiling with ghc 7.10 and higher and haskell-mpfr when compiling with ghc 7.8.

There is also an Integer backend which does not need MPFR, but it is not yet complete.
