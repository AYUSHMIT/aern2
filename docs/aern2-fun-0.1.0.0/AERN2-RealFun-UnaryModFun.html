data UnaryModFun Source #

Here we represent a continuous function \(f\) using its evaluation function on dyadics and a "local" modulus of continuity \(\omega\) in the following sense: . For any dyadic \(d_1,d_2\in b\) with \(|d_1-d_2|\leq 2^{ -\omega(b)(i)}\) we have \(|f(d_1)-f(d_2)| \leq 2^{ -i }\).

For any \(b\), the modulus of continuity has to be a non-decreasing function: \[ i\leq j \implies \omega(b)(i)\leq\omega(b)(j) \] and \(omega(b)(i)\) has to converge to \(\infty\) with increasing \(i\).
