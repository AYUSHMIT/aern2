-> MPFloat

-> MPBall -> MPBall

f on MPBall rounding *outwards*

Computes a real function f from correctly rounded MPFR-approximations and a number lip which is a Lipschitz constant for f, i.e. |f(x) - f(y)| <= lip * |x - y| for all x,y.
